# Poker
# 五張撲克牌遊戲

## 一、專案簡介

本專案是一個使用 C# Windows Forms 製作的五張撲克牌遊戲。玩家可以先輸入押注金額，再進行發牌、換牌與判斷牌型。程式會根據玩家最後的牌型判斷結果，搭配不同牌型的賠率，計算中獎金額並更新玩家目前的總資金。

本專案主要練習 Windows Forms 介面設計、Button 事件處理、PictureBox 顯示圖片、陣列操作、隨機洗牌、撲克牌牌型判斷，以及下注金額與賠率計算等功能。

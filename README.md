# Poker

## 一、專案簡介

玩家可以先輸入押注金額，再進行發牌、換牌與判斷牌型。程式會根據玩家最後的牌型判斷結果，搭配不同牌型的賠率，計算中獎金額並更新玩家目前的總資金。

---
## 二、主要功能

### 1. 下注功能

玩家在每一局開始前，需要先輸入押注金額，並按下「押注」按鈕。

押注時，程式會檢查：

- 押注金額是否為數字
- 押注金額是否大於 0
- 押注金額是否超過目前總資金

押注成功後，系統會先從總資金扣除押注金額，並啟用發牌按鈕。
